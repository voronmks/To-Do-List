# 📝 To-Do List

Simple task management examples written in multiple programming languages.

---

## 📖 Overview

This repository demonstrates basic To-Do List implementations in:

- JavaScript
- Python
- C#

The project is intended for educational purposes and beginner practice.

---

## ✨ Features

- Add tasks
- Store tasks in memory
- Display task list
- Simple and readable code

---

## 📂 Project Structure

```text
javascript/
python/
csharp/
```

## Example Output

```text

```

## Running

### JavaScript

```bash
node todo.js
```

### Python

```bash
python todo.py
```

### C#

```bash
dotnet run
```
